Firebase設定の取得方法
① Firebase Console でプロジェクトを作成
② プロジェクト設定 → 「アプリを追加」→ ウェブ (＜/＞) を選択
③ 「Firebase SDK の追加」に表示される firebaseConfig の
　 { ... } 部分をコピーして以下に貼り付けてください
④ Firestore Database を作成し、テストモードで開始してください

🔑 テーマと問い

作品の骨格となる「問い」と「テーマ」を言語化しましょう。

この作品が問いかける、核心的な「問い」は何ですか？
この物語を貫くテーマ・概念を一言で表すとしたら？
読者に一緒に考えてほしいこと、問いかけたいことは？
自由メモ（参考作品・引用・インスピレーションなど）
💚 届けたい感情・体験

読者の心と体に刻みたい感情・体験を具体的に描き出しましょう。

読者に感じてほしい感情は何ですか？（3つまで）
読書中、どんな体験をしてほしいですか？
本を閉じた後、読者の心に残したいものは？
自由メモ
🌀 自分の核心・創作動機

なぜあなたが書くのか。その根っこを掘り下げましょう。

なぜこの物語を、あなたが書くのですか？
この作品と、あなた自身との繋がりは？
あなたにしか書けない、この作品の「固有性」とは？
自由メモ（衝動・原風景・忘れたくない言葉など）
📮 理想の読者像

この本を手に取る「あの人」の姿を、鮮明に思い描きましょう。

一番届けたい人は、どんな人ですか？（具体的に）
その人が抱えている悩み・渇望・問いは何ですか？
この本を読んだ後、その人の何が変わってほしいですか？
その読者への手紙（自由に書いてみてください）
